# Organizers — pixel-art portraits (TEXT-ONLY, no reference photos)

**Model:** Nano Banana Pro · **Aspect:** 1:1 · **Resolution:** 1k (1024², keeps pixels chunky — confirmed) · **No reference image**.

**Read this first — how to get the chunky pixels:** generate at **1K (1024×1024)**, NOT 4k — the
lower resolution is what makes Nano Banana output true chunky pixels (4k came out as a smooth
detailed illustration). Optionally still run the `scripts/pixelate.py` pass afterwards to unify the
palette across all five and match the fighter roster exactly — but at 1K the raw output is already
close.

Backdrop (chosen): a **shadowed ancient stone temple** — tall dark column silhouettes receding into
mist, cool blue rim-light + a faint warm torch glow deep behind, moody cinematic depth. Same for all
five. Not the dungeon brick wall, not the neon grid. Fixed tight framing.

Output raw → `docs/the organizers/raw/`, pixelated → `public/assets/demo/organizers/crew-0X.png`.
